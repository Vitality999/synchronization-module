Инструкция по запуску
---------------------------------------

Убедиться, что установлен Python 2.7:

python --version

Скрипт запуска: 

python sql_parser.py sql_file_name

Команду надо запускать из директории с файлами -py и -sql. 

Требуется разработать утилиту на python sql_parser.py принимающую как входной параметр название sql файла.

 
Описание работы утилиты
---------------------------------------
Из файла выделить следующие параметры:

-название таблицы
-слой таблицы
-код СИ (6-значный)

и выполняющую парсинг скрипта загрузки таблицы stg_dds.

Результат парсинга  укладывать в файл (добавлением в начало файла лога парсинга):

etl_parser.log 

 

/* <Название файла> <Название целевой таблицы> <Код СИ начала диапазона> <Код СИ окончания диапазона>  <Дата> <Время> */

Содержимое файла:

список ODS таблиц (через ; ), влияющих на загрузку STG_DDS таблицы
список STG_DDS таблиц (через ; ), влияющих на загрузку STG_DDS таблицы
список DDS таблиц (через ; ), влияющих на загрузку STG_ADS таблицы
список STG_ADS таблиц (через ; ), влияющих на загрузку STG_ADS таблицы
