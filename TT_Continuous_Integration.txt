Утилита сборки и раскатки скриптов через Jenkins

1) Jenkins запускает утилиту сборки патча на python, собирается некий патч и коммитится в svn
2) Выполняется проверка корректности наполнения патча
3) по коммиту в svn файлы патча подтягиваются в Jenkins
4) идет раскатка патча на тестовой среде с помощью утилиты на python
5) запускаются исполняющие скрипты патча и проверка качества их работы 
6) выполняется нотификация о успешности/неуспешности работы

Тип скриптов:
- python
- sql
